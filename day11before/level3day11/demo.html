安装方法

安装node(.msi)

在cmd中 输入 npm install gulp -g

如果不放心 可以gulp(爆出没有localversion...)

在输入 npm install gulp

创建项目文件目录

cmd 声明指定当前项目目录 npm init

回车 直到输入yes

npm install gulp --save-dev

在项目目录创建gulpfile.js

最后 书写 default 任务

-----------------------------

tomcat 是后端的服务器 用node进行模拟

gulp-connect:连接 建立服务器端口
